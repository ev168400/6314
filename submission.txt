Story number and name: 1 - Extend User profile detail with usage
Story points: 3
Brief description to see the story in action:
Visit user profile page to view most recent post and most commented on post.
Thumbnails link to photo and scroll to destination. 
If no photos exist for a user, an appropriate message is shown.
If no photos have any comments, the most recent photo will be shown, 
but an appropriate message is shown for highest commented photo.


Story number and name: 2 - @mentions in comments
Story points: 6
Brief description to see the story in action:
Visit user photo album page, and add a comment to a photo.
@mentions can be created these three ways @first_name, @last_name, or @first_nameLast_name.
@mentions are linked to the mentioned users profile page.
If an @mention is made for a user who does not have a profile, no link will be made.
Once on the mentioned user profile page, a section of all mentions can be found.
Each mention will include, name of commenter, name of post owner, the comment, and a thumbnail of the photo.
The thumbnail can be clicked to be directed to photo location on photo album page. 


Story number and name: 5 - Deleting comments, photos, and users
Story points: 4
Brief description to see the story in action:
Visit logged in users photo album page, or any post where logged in user has made a comment.
Any post or comment created by the logged in user will have a small trash can icon in top right.
Click the trash can to delete the comment or post. 
Navigate to logged in user profile page where a 'delete profile' button can be found in the bottom right.
Click the button, a warning will appear ensuring user wants delete profile.
Select delete to delete page, will be redirected to login page.
Select cancel if you do not want to delete page.


Story number and name: 8 - sidebar list marks users with new activity
Story points: 3
Brief description to see the story in action:
Log in to app and look at sidebar on left side. 
Logged in user will be at top, friends will be below. 
Each user will show their most recent activity.
There are 5 activites: log in, log out, comment on a post, post a photo, register a new profile.
If the most recent activity is create a new post, a hyperlinked thumbnail will be present.
Select the thumbnail to be redirected to photos location in photo album.
